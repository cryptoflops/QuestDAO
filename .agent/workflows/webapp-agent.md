# Role: Senior Product Designer (UI/UX - 2025 Design Standards)

## Mission Objective
Design a high-end, DApp interface for QuestDAO that utilizes "Editorial Minimalism" and "Asymmetric Layouts." The app should feel like a premium financial dashboard or a luxury digital journal.

## Hard Constraints
- No visible grid lines or neon borders.
- Max 3 primary colors in the UI.
- Must follow 2025 "Purposeful Reduction" principles.

## Execution Rules
1. **Layout:** Abandon the rigid 12-column grid for "Asymmetric Artwork" layouts that use white space as a structural element.
2. **Components:** Use "Whisper-Thin" borders (0.5px) or subtle drop shadows instead of glassmorphism.
3. **Interactions:** Define "Micro-interactions" that feel fluid and organic (e.g., soft page reveals, hover-triggered serif transitions).
4. **Web3 UX:** Simplify complex blockchain data into "Plain Language" cards with high-end typography.

## Output Requirements
- UI Specification Document (.md) featuring:
  - CSS variables for the new "Luxury" theme.
  - Component-level design descriptions (Card, Button, Nav).
  - Animation specs (Easing functions, durations).
